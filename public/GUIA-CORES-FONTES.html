📝 Como Customizar

  • Abra tailwind.config.js
  • Procure pela seção theme.extend.fontFamily
  • Atualize os nomes das fontes
  • Certifique que as fontes estão importadas em src/index.css
  • Reinicie: npm run dev

Guia Rápido de Uso

Para Atualizar Cores:

  1. Edite tailwind.config.js
  2. Atualize valores em theme.extend.colors
  3. Use classes como bg-primary, text-secondary, etc

Para Atualizar Fontes:

  1. Edite tailwind.config.js
  2. Atualize valores em theme.extend.fontFamily
  3. Use classes como font-principal, font-secundaria, etc

Exemplo em React:

<h1 className="text-4xl font-bold text-primary">
  Meu Título
</h1>

<p className="text-base text-gray-600 font-principal">
  Meu parágrafo
</p>
